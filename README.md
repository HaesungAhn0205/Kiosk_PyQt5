# 키오스크 점자 보조기 설계 및 제작 
### Design and Fabrication of Braille Assistant for Interactive Kiosk

본 프로젝트는 편의시설 환경에서 급격히 보편화되고 있는 인터랙티브 키오스크(Interactive Kiosk)에서 소외되는 시각장애인의 디지털 접근성을 향상하기 위해 개발된 **'키오스크 점자 보조기'** 시스템입니다. 파이썬(Python) 및 PyQt5 기반의 키오스크 프로그램과 라즈베리파이(Raspberry Pi)의 GPIO 제어를 결합하여, 시각장애인이 기존의 음성 안내뿐만 아니라 실시간 물리 점자 출력을 통해 키오스크를 직관적이고 독립적으로 조작할 수 있도록 지원합니다.

---

## 🛠️ 주요 기능 (Key Features)

* **초음파 센서 기반 자동 안내 (TTS)**
  * 사용자가 키오스크 전방 50cm 이내로 접근하면 초음파 센서가 이를 감지합니다.
  * 외부 스피커(Aux 연결)를 통해 기기 구성 및 하드웨어 버튼 조작 방법에 대한 안내 음성을 자동으로 출력합니다.
* **물리 스위치 버튼 인터페이스**
  * 터치스크린 조작이 어려운 시각장애인을 위해 총 7개의 하드웨어 스위치 버튼을 배치하여 명확하고 직관적인 입력을 지원합니다.
* **실시간 18구 하드웨어 점자 표시**
  * 총 18개의 솔레노이드 액츄에이터를 병렬 제어하여 메뉴 이름을 실시간 점자 데이터로 표현합니다.
  * 각 글자는 사용자의 가독성을 고려해 2초 동안 출력된 후 다음 글자로 전환되며, 출력이 종료되면 모든 모터가 자동으로 초기화됩니다.

## 🔌 하드웨어 사양 및 아키텍처 (Hardware Architecture)

점자 보조기 시스템의 하드웨어 회로는 라즈베리파이를 중심으로 병렬 입력 및 전원 분리형 구동 회로로 구성되어 있습니다.

* **메인 제어 장치:** Raspberry Pi 3B+
* **점자 구동 모터:** 솔레노이드 액츄에이터 (DS-0420S) 18개
* **센서 및 전동 제어:** * 초음파 센서 (HC-SR04)
  * 릴레이 모듈 (라즈베리파이 신호로 외부 고전류 전원을 온/오프 제어)
  * 단자대 (Terminal Block)를 통한 배선 효율화
* **회로 보호 장치:** 솔레노이드의 인덕턴스 특성에 따른 플라이백(Flyback) 고전압 현상을 방지하고, 라즈베리파이 및 시스템 오작동을 차단하기 위해 솔레노이드 양단에 역방향 다이오드 연결
* **구동 전원:** 18개 모터의 동시 작동 전류를 안정적으로 공급하기 위해 외장형 DC 어댑터 (입력 AC 220V / 출력 DC 12V 10A) 배치
* **기구물 외관 케이스:** 3D 프린팅 기법으로 정밀 설계된 백색 상판과 직접 가공한 백색 아크릴 판(측면, 하판, 내부 거치대)의 조합으로 구동축 간섭 최소화 및 내부 회로 보호

### 📌 하드웨어 핀 매핑 (Pin Mapping)

| 구분 | 컴포넌트 기능 | BCM 핀 번호 | 물리적 연결 형태 및 사양 |
| :--- | :--- | :--- | :--- |
| **센서** | 초음파 센서 Trigger | `GPIO 1` | 초음파 송신 신호 출력 |
| | 초음파 센서 Echo | `GPIO 0` | 전압 분배 회로(Voltage Divider) 거쳐 입력 |
| **버튼 (입력)** | 이전 메뉴 탐색 (Prev) | `GPIO 22` | 내부 풀업 저항(Pull-Up) 사용, Falling Edge 감지 |
| | 다음 메뉴 탐색 (Next) | `GPIO 27` | 내부 풀업 저항(Pull-Up) 사용, Falling Edge 감지 |
| | 메뉴 장바구니 추가 (Press) | `GPIO 17` | 내부 풀업 저항(Pull-Up) 사용, Falling Edge 감지 |
| | 점자 출력 실행 (Braille) | `GPIO 4` | 내부 풀업 저항(Pull-Up) 사용, Falling Edge 감지 |
| | 장바구니 합산 (Total) | `GPIO 18` | 내부 풀업 저항(Pull-Up) 사용, Falling Edge 감지 |
| | 장바구니 초기화 (Clear) | `GPIO 15` | 내부 풀업 저항(Pull-Up) 사용, Falling Edge 감지 |
| | 최종 결제/주문 (Payment) | `GPIO 14` | 내부 풀업 저항(Pull-Up) 사용, Falling Edge 감지 |
| **모터 (출력)** | 솔레노이드 핀 (총 18개) | `26, 19, 13, 6, 5, 11, 9, 10, 21, 20, 16, 12, 7, 8, 25, 24, 3, 2` | 릴레이 모듈 단자와 1:1 매핑되어 HIGH/LOW 제어 |
